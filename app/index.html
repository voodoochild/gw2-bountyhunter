Black Lion Chest

Bounty Hunter

Shows whether or not a bounty is in combat on your chosen World.

Automatically updates every 20 seconds. Orange swords are shown over bounties currently in combat, like so.

For help finding a bounty, look no further than Dulfy’s excellent guide.

The code for this web app is up on Github if you want to see how it works.

Follow @blacklionchest for updates and information.
